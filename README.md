# Introduction to Programming for Business - TP1

Repository ini berisi latihan dan jawaban Tugas Personal (TP1) mata kuliah Introduction to Programming for Business.

## Daftar File
1. **PrimitiveExample.java**: Contoh tipe data primitif.
2. **NonPrimitiveExample.java**: Contoh tipe data non-primitif (String & Array).
3. **Calculator.java**: Contoh penggunaan static method.
4. **Mahasiswa.java**: Contoh Kelas, Objek, dan Atribut.
5. **NonStatic.java**: Contoh penggunaan non-static method (melalui objek).
6. **JenisKendaraanIfElse.java**: Contoh pernyataan kontrol IF-ELSE.
7. **JenisKendaraanSwitch.java**: Contoh pernyataan kontrol SWITCH-CASE.
8. **ContohArrayList.java**: Contoh penggunaan ArrayList.
9. **TwoSumSolver.java**: Jawaban Pertanyaan 5 (Mencari indeks dari dua angka yang berjumlah sesuai target).
10. **PalindromeChecker.java**: Jawaban Pertanyaan 6 (Memeriksa apakah angka adalah palindrom).

## Cara Menjalankan
1. Pastikan Java (JDK) sudah terinstal.
2. Kompilasi: `javac NamaFile.java`
3. Jalankan: `java NamaFile`
